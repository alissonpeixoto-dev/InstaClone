# 🔧 Como Ativar "Salvar Posts"

## Passo 1: Abra o Supabase
https://supabase.com/dashboard

## Passo 2: SQL Editor
1. No menu lateral, clique em **SQL Editor**
2. Clique em **New Query** (ou o botão **+**)

## Passo 3: Copie o SQL
1. Abra o arquivo `SETUP_FAVORITES.sql` neste projeto
2. **Copie TODO o conteúdo**
3. Cole no editor do Supabase

## Passo 4: Execute
Clique no botão **▶ RUN** (verde, no canto superior direito)

## Pronto! ✅
- Volte ao app React
- Pressione **F5** para recarregar
- Tente salvar um post
- Deve funcionar agora!

---

## Se não funcionar:
1. Abra o Console (F12)
2. Tente salvar um post
3. Procure por mensagens `[Favorites]` no console
4. Copie qualquer mensagem de erro e compartilhe
